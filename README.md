### This branch is currently under development
This branch is for teting and developing dev encyclopedia using AstroJS framework. The site is deployed in: [https://astro-devpedia.pages.dev](https://astro-devpedia.pages.dev/)

Once completed, this branch would be merged to main branch and deployed on the main site. For now, you can contribute by adding markdown files to this repository (view examples on content / terms folder)
